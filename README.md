# KSS Boilerplate

This is a boilerplate for a [KSS Living Styleguide](https://github.com/kneath/kss) that contains markup for the following:

- base typography
- colour swatches
- buttons
- navigation
- and more!

Run it with:

    bundle install
    bundle exec ruby app.rb
